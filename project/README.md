# Project

Container for all project-related documentation, inputs, and outputs that drive the AI-assisted SDLC for this repository.

## Layout

- `inputs/` — source material fed into the SDLC: briefs, requirements, research notes, transcripts, references, raw artifacts from stakeholders.
- `outputs/` — artifacts produced by the SDLC: specs, designs, ADRs, plans, generated documentation, review notes.

Treat `inputs/` as mostly read-only context and `outputs/` as the working surface where AI agents and humans collaborate.
